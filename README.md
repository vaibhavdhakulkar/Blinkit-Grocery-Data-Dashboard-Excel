# Blinkit-Grocery-Data-Dashboard-Excel
Blinkit Grocery Data Dashboard (Excel) – A comprehensive dashboard analyzing Blinkit’s grocery sales. It visualizes total and average sales by outlet and year, compares fat content, outlet size, and item types, highlights top categories like Fruits &amp; Vegetables and Snack Foods, and tracks outlet growth trends using Excel, Pivot Tables, and Charts.

Key Highlights:
Visualized total and average sales across outlets and years
Compared sales by fat content, outlet size, and item type
Identified top-performing categories like Fruits & Vegetables and Snack Foods
Tracked growth patterns through outlet establishment analysis
Skills & Tools: Excel, Pivot Tables, Charts, Data Cleaning, Visualization
